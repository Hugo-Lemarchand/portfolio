📦 Installation OCS Inventory

Open Computers and Software Inventory - Inventaire automatique du parc

📋 Prérequis

  • Serveur Debian/Ubuntu
  • Apache2 avec mod_perl
  • MariaDB/MySQL
  • PHP 7.4+ avec extensions
  • Perl et modules CPAN
  • Minimum 2 Go RAM, 10 Go disque

🔧 Installation des dépendances

1Mise à jour et installation des paquets de base

sudo apt update && sudo apt upgrade -y
sudo apt install -y apache2 libapache2-mod-perl2 mariadb-server
sudo apt install -y php php-{mysql,gd,mbstring,xml,curl,zip}
sudo apt install -y perl libxml-simple-perl libdbi-perl libdbd-mysql-perl libapache-dbi-perl libnet-ip-perl libsoap-lite-perl libarchive-zip-perl make build-essential

2Installer les modules Perl additionnels

sudo cpan install XML::Entities Apache2::SOAP
sudo cpan install Mojolicious::Lite Plack::Handler Switch Archive::Zip

3Créer la base de données

sudo mysql -u root -p
CREATE DATABASE ocsweb;
CREATE USER 'ocs_user'@'localhost' IDENTIFIED BY 'MotDePasseOCS123!';
GRANT ALL PRIVILEGES ON ocsweb.* TO 'ocs_user'@'localhost';
FLUSH PRIVILEGES;
EXIT;

📥 Installation du serveur OCS

4Télécharger OCS Inventory Server

cd /tmp
wget [github.com](https://github.com/OCSInventory-NG/OCSInventory-ocsreports/releases/download/2.12.1/OCSNG_UNIX_SERVER-2.12.1.tar.gz)
tar -xvzf OCSNG_UNIX_SERVER-2.12.1.tar.gz
cd OCSNG_UNIX_SERVER-2.12.1

5Lancer l'installation

sudo ./setup.sh

Répondre aux questions de l'assistant :

  • Apache config directory : /etc/apache2/conf-available
  • Apache user/group : www-data
  • Perl binary : /usr/bin/perl
  • Logfile directory : /var/log/ocsinventory-server

6Activer la configuration Apache

sudo a2enconf ocsinventory-reports
sudo a2enconf z-ocsinventory-server

# Activer les modules nécessaires
sudo a2enmod perl
sudo a2enmod rewrite

# Redémarrer Apache
sudo systemctl restart apache2

7Configurer les permissions

sudo chown -R www-data:www-data /var/lib/ocsinventory-reports
sudo chmod -R 755 /var/lib/ocsinventory-reports

sudo mkdir -p /var/log/ocsinventory-server
sudo chown www-data:www-data /var/log/ocsinventory-server

🖥️ Configuration via l'interface web

8Accéder à l'installation web

[ip_du_serveur](http://IP_DU_SERVEUR/ocsreports/install.php)

9Configurer la connexion

  • MySQL server : localhost
  • MySQL user : ocs_user
  • MySQL password : MotDePasseOCS123!
  • MySQL database : ocsweb
⚠️ Important : Après l'installation, supprimer le fichier install.php ! 
sudo rm /usr/share/ocsinventory-reports/ocsreports/install.php

Identifiants par défaut

  • Login : admin
  • Mot de passe : admin

💻 Installation de l'agent Windows

1Télécharger l'agent

Depuis le GitHub officiel : OCSInventory-NG/OCSInventory-Windows-Agent

2Installation silencieuse

OCS-Windows-Agent-Setup.exe /S /SERVER=[ip_serveur](http://IP_SERVEUR/ocsinventory)

3Configuration via fichier

Fichier de config : C:\ProgramData\OCS Inventory NG\Agent\ocsinventory.ini

[OCS Inventory Agent]
Server=[ip_serveur](http://IP_SERVEUR/ocsinventory)
SSL=0
TAG=MonEntreprise

🐧 Installation de l'agent Linux

# Installer les dépendances
sudo apt install -y libmodule-install-perl dmidecode libxml-simple-perl libcompress-zlib-perl libnet-ip-perl libwww-perl libdigest-md5-perl libdata-uuid-perl

# Télécharger l'agent
cd /tmp
wget [github.com](https://github.com/OCSInventory-NG/UnixAgent/releases/download/v2.10.0/Ocsinventory-Unix-Agent-2.10.0.tar.gz)
tar -xvzf Ocsinventory-Unix-Agent-2.10.0.tar.gz
cd Ocsinventory-Unix-Agent-2.10.0

# Installer
perl Makefile.PL
make
sudo make install

# Configurer
sudo ocsinventory-agent --server [ip_serveur](http://IP_SERVEUR/ocsinventory) --tag Linux-Servers

# Automatiser avec cron
echo "0 12 * * * root /usr/local/bin/ocsinventory-agent" | sudo tee /etc/cron.d/ocsinventory
