🐧 Installation Debian

Système d'exploitation GNU/Linux stable et fiable

📋 Prérequis

Composant Minimum Recommandé
RAM 512 Mo 2 Go+
Espace disque 10 Go 20 Go+
Processeur 1 GHz 2 GHz+
  • Clé USB de 4 Go minimum
  • Image ISO Debian (netinst recommandée) depuis debian.org

🔧 Étapes d'installation

1Création de la clé USB bootable

# Windows : utiliser Rufus
# Linux :
sudo dd if=debian-12.*-amd64-netinst.iso of=/dev/sdX bs=4M status=progress sync

2Démarrage et sélection du mode

  • Booter sur la clé USB
  • Choisir "Graphical Install" ou "Install" (mode texte)

3Configuration de base

  • Langue : Français
  • Pays : France
  • Clavier : Français (azerty)

4Configuration réseau

Nom de la machine : debian-server
Domaine : local (ou laisser vide)

5Configuration des utilisateurs

  • Mot de passe root (laisser vide pour désactiver root et utiliser sudo)
  • Créer un utilisateur standard

6Partitionnement du disque

Pour les débutants :

  • Choisir "Assisté - utiliser un disque entier"
  • Sélectionner "Tous les fichiers dans une seule partition"
💡 Configuration avancée : 
/boot   - 500 Mo (ext4)
/       - 20 Go (ext4)
/home   - Reste du disque (ext4)
swap    - 2x RAM (max 8 Go)

7Sélection des logiciels

Cocher selon vos besoins :

  • ☐ Environnement de bureau Debian (GNOME, KDE, XFCE...)
  • ☑ Serveur SSH
  • ☑ Utilitaires usuels du système

8Installation de GRUB

  • Installer GRUB sur le disque principal
  • Sélectionner /dev/sda

⚙️ Configuration post-installation

Mise à jour du système

sudo apt update
sudo apt upgrade -y
sudo apt full-upgrade -y

Installation des outils essentiels

sudo apt install -y vim curl wget git htop net-tools

Configuration de sudo (si root désactivé)

# Ajouter l'utilisateur au groupe sudo
su -
usermod -aG sudo votre_utilisateur
# Déconnecter et reconnecter

Configuration SSH

# Éditer la configuration
sudo nano /etc/ssh/sshd_config

# Recommandations sécurité :
PermitRootLogin no
PasswordAuthentication yes
Port 22

# Redémarrer SSH
sudo systemctl restart sshd

Configuration du pare-feu (UFW)

sudo apt install ufw
sudo ufw allow ssh
sudo ufw enable
sudo ufw status

📚 Commandes essentielles

# Gestion des paquets
apt search <paquet>
apt install <paquet>
apt remove <paquet>
apt autoremove

# Informations système
uname -a
lsb_release -a
df -h
free -h

# Gestion des services
systemctl status <service>
systemctl start/stop/restart <service>
systemctl enable/disable <service>

🔐 Sécurisation de base

  • Mettre à jour régulièrement le système
  • Configurer les mises à jour automatiques de sécurité
  • Désactiver les services inutiles
  • Configurer fail2ban contre les attaques brute force
# Installation de fail2ban
sudo apt install fail2ban
sudo systemctl enable fail2ban
sudo systemctl start fail2ban
