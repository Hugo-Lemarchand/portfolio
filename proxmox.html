🖥️ Installation Proxmox VE

Hyperviseur de virtualisation open-source

📋 Prérequis

  • Un serveur ou PC dédié avec processeur supportant la virtualisation (VT-x/AMD-V)
  • Minimum 4 Go de RAM (8 Go+ recommandé)
  • Disque dur de 32 Go minimum
  • Clé USB de 8 Go minimum pour l'installation
  • Image ISO de Proxmox VE (téléchargeable sur proxmox.com)

🔧 Étapes d'installation

1Création de la clé USB bootable

Télécharger l'ISO depuis le site officiel et utiliser un outil comme Rufus ou Balena Etcher :

# Sous Linux avec dd
sudo dd if=proxmox-ve_*.iso of=/dev/sdX bs=4M status=progress

2Démarrage sur la clé USB

  • Configurer le BIOS/UEFI pour démarrer sur la clé USB
  • Activer la virtualisation (VT-x ou AMD-V) dans le BIOS
  • Sélectionner "Install Proxmox VE"

3Configuration de l'installation

  • Accepter la licence EULA
  • Sélectionner le disque cible pour l'installation
  • Configurer le pays, le fuseau horaire et la disposition clavier
  • Définir le mot de passe root et l'email administrateur

4Configuration réseau

Hostname: proxmox.local
IP Address: 192.168.1.100
Netmask: 255.255.255.0
Gateway: 192.168.1.1
DNS Server: 192.168.1.1

5Finalisation

Après l'installation, accéder à l'interface web :

[192.168.1.100](https://192.168.1.100:8006)

⚙️ Configuration post-installation

Désactiver le dépôt Enterprise (optionnel)

# Éditer le fichier sources
nano /etc/apt/sources.list.d/pve-enterprise.list
# Commenter la ligne existante

# Ajouter le dépôt no-subscription
echo "deb [download.proxmox.com](http://download.proxmox.com/debian/pve) bookworm pve-no-subscription" > /etc/apt/sources.list.d/pve-no-subscription.list

# Mettre à jour
apt update && apt full-upgrade -y

Créer un stockage pour les VMs

  • Datacenter → Storage → Add
  • Choisir le type (Directory, LVM, ZFS...)
  • Configurer le chemin et les contenus autorisés
💡 Conseil : Utiliser ZFS pour bénéficier des snapshots et de la réplication.

🚀 Création de votre première VM

  1. Cliquer sur "Create VM" en haut à droite
  2. Configurer l'ID et le nom de la VM
  3. Sélectionner l'image ISO
  4. Configurer le disque virtuel (taille, format qcow2/raw)
  5. Allouer CPU et RAM
  6. Configurer le réseau (bridge vmbr0)
  7. Valider et démarrer la VM

📚 Commandes utiles

# Lister les VMs
qm list

# Démarrer une VM
qm start <vmid>

# Arrêter une VM
qm stop <vmid>

# Créer un snapshot
qm snapshot <vmid> <snapname>

# Voir les logs
journalctl -u pvedaemon
